树

树的定义

  • 树是由若干元素组成的有层次关系的拓扑结构
    • 结构中的每个元素称为树的结点
    • 结构中连接两个结点的关系称为树的边
  • 常用于表示从属关系，族谱等

树的相关概念

  • 父节点：上一层中与此结点相邻的结点
  • 子节点：下一层中与此结点相邻的结点
  • 根节点：没有父节点的结点
  • 叶子结点：没有子节点的结点
  • 祖先节点：从父节点到根结点的路径上所有结点
  • 子孙结点：以该节点为祖先节点的所有结点
  • 子树：以该结点为根结点，包括其全部子孙节点，以及连接他们所有的边构成的树形结构
  • 结点的深度：以根节点的深度为0，其他结点的深度为父节点+1
  • 树的高度：树中所有节点的最大深度

树的性质

  • 一棵N个结点的树恰好有N-1个结点
  • 树上任意两结点之间有且只有一条路径相连

树的分类

  • 有根树&无根树（不关心树的层次关系）
  • 一般树&二叉树

树的表示法

  • 子结点表示法
    • 利用vector或邻接表存储每个节点的全部子节点，
    • 利用父节点找到子节点(知道根节点)
    • 利用父节点找到子节点(不知道根节点)
  • 邻接关系表示法
    • 利用vector存储每个节点相邻的结点

树的遍历方式

  • DFS遍历（深度优先遍历）
  • BFS遍历（广度优先遍历）

二叉树

定义

  • 对于每个结点，子节点的个数均不超过二的树

  • 子节点分为左子节点与右子节点两种

  • 对应的，有左子树和右子树两种概念

性质

  • 深度为d的结点至多有$2^{d-1}$个
  • 高度为h的二叉树至多有$2^h-1$个结点，此时此二叉树为满二叉树

分类

  • 满二叉树
  • 完全二叉树
    • 定义：若一完全二叉树深度为k，则前k-1层组成一个满二叉树，第k层结点从左到右
    • 性质：若将结点从左到右，从上到下标号，则
      • l[i] = i * 2;
      • r[i] = i * 2 + 1;

遍历方式

先序遍历

  • 先访问根节点，再访问左子树，最后访问右子树

中序遍历

  • 先访问左子树，再访问根节点，最后访问右子树

后序遍历

  • 先访问左子树，再访问右子树，最后访问根节点

  • 代码（先序）

并查集

定义

  • 一种可以动态维护若干个不重叠的集合，并支持合并和查询的数据结构
  • 每个集合本质上是一棵树

基本操作

  1. find：查询某个结点
  2. merge：合并两个集合

表示方法

  • 代表元法
  • #include<iostream>
using namespace std;
int p[maxn];
int n;
int find(int u){
    if(p[u] == u){
        return u;
    }
    return p[u] = find(p[u]);
}
void merge(int u, int v){
    u = find(u);
    v = find(v);
    if(u == v){
        return;
    }
    p[u] = v;
}
void init(int n){
    for(int i = 1; i <= n; i++){
        p[i] = i;
    }
}
