运算

注意：除赋值外，其他运算表达式均从左到右

算术运算符

运算符 描述 例子(A = 20,B = 10)
+ 把两个数相加（废话 A + B = 30
- 把两个数相减（废话 A - B = 10
* 把两个数相乘（废话 A * B = 200
/ 把两个数相除（废话 A / B = 2
% 取模（整数后的余数 A % B = 0
++ 自增运算符 A++ = 21
– 自减运算符 B– = 9

关系运算符

运算符 描述 例子(A = 20,B = 10)
== 检查两个操作数的值是否相等，如果相等则条件为真 A == B false
!= 检查两个操作数的值是否不相等，如果不相等则条件为真 A != B true
> 检查左操作数的值是否大于右操作数的值，如果是则条件为真 A > B true
< 检查左操作数的值是否小于右操作数的值，如果是则条件为真 A < B false
>= 检查左操作数的值是否大于或等于右操作数的值，如果是则条件为真 A >= B true
<= 检查左操作数的值是否小于或等于右操作数的值，如果是则条件为真 A <= B false

逻辑表达式

运算符 描述 例子(A = true,B = false)
&& 称为逻辑与运算符。如果两个操作数都非零，则条件为真 A && B false
|| 称为逻辑或运算符。如果两个操作数中有任意一个非零，则条件为真 A || B true
！ 称为逻辑非运算符。用来逆转操作数的逻辑状态。如果条件为真则逻辑非运算符将使其为假 !(A && B) true

位运算符

运算符 描述 实例(A = 00111100，B = 00001101)
& 按位与。二进制下，均为1则为1，其他均为0 A & B = 0000 1100
| 按位或。二进制下，有1则为1 A | B = 0011 1101
^ 按位异或。二进制下，均为0则为0，其他均为1 A ^ B = 0011 0001
~ 按位取反。二进制下，0变成1，1变成0 A ~ B = 1100 0011
<< 左移运算符。二进制下左移，空出的用0补充(n << m = n × 2^m^) A << 2 = 1111 0000
>> 右移运算符。二进制下右移，移出的被舍弃(n << m = n ÷ 2^m^) A >> 2 = 0000 1111

注意：如果进行有关于2^n^的乘除操作，使用<<后>>最快(因为计算机内部数字采用二进制存储)

赋值运算符

运算符 描述 实例(A = 1，B = 2，C)
= 将右边操作数的值赋给左边 C = A + B C = 3
+=，-=，*=，/=，%=，<<=，>>=，&=，^=，!= 。。。见上 。。。

常用函数
setw()

这个函数用于控制输出的宽度

如果在多个cout语句中使用相同的setw()，并使用setiosflags(ios::right)（见下），可以实现各行数据右对齐，如果指定相同的精度，可以实现上下小数点对齐。

setprecision()

控制输出小数的位数

setiosflags()

控制左右对齐

总结
控制符 作 用
dec 设置数值的基数为10
hex 设置数值的基数为16
oct 设置数值的基数为8
setfill(c) 设置填充字符c，c可以是字符常量或字符变量
setprecision(n) 设置浮点数的精度为n位。在以一般十进制小数形式输出时，n代表有效数字。在以fixed(固定小数位数)形式和 scientific(指数)形式输出时，n为小数位数
setw(n) 设置字段宽度为n位
setiosflags( ios::fixed) 设置浮点数以固定的小数位数显示
setiosftags( ios::scientific) 设置浮点数以科学记数法(即指数形式)显示
setiosflags( ios::left) 输出数据左对齐
setiosflags( ios::right) 输出数据右对齐
setiosflags( ios::skipws) 忽略前导的空格
setiosflags( ios::uppercase) 数据以十六进制形式输出时字母以大写表示
setiosflags( ios::lowercase) 数据以十六进制形式输出时宇母以小写表示

STL(标准模板库)

vector(弹性数组)

性质

vector(向量)是一个能够存放任意类型的动态数组

定义
访问

访问可以直接利用数组或使用迭代器

使用
添加

添加通过push_back实现

push_back：向量尾部增加一个元素

删除

删除通过pop_back，erase或clear实现

pop_back：删除向量中最后一个元素

注意：pop_back虽然是删除，但是全部遍历的时候仍然存在，只是通过.size()返回是删除后的长度

clear：删除向量的全部

注意：clear与pop_back类似，虽然是删除，但是全部遍历的时候仍然存在，只是通过.size()返回是删除后的长度

erase：删除向量中迭代器指向元素

格式：erase(迭代器)

注意：erase完后的vector会更新

判断

使用empty()判断vector是否为空

排序

排序同数组排序方式一致，可使用sort

queue(队列)

性质

队列是一种特殊的线性表，特殊之处在于：它只允许在表的前端进行删除操作，只允许在表的后端进行插入操作

定义
访问
一、通过头和尾访问
二、使用类似数组的方式访问
使用
添加

使用push，用法同vector的push_back

删除

使用pop或clear，用法同vector的pop_back或clear

判断

使用empty()，用法同vector的empty()

删除

queue无法完成删除任务，但可以用优先队列实现

deque(双向队列)

性质

deque是双端队列容器，也就是可以在两头添加元素的队列

定义
访问
一、通过头和尾访问
二、使用类似数组的方式访问
使用
添加

使用push_back，用法同vector的push_back

删除

使用pop_back，pop_front或clear，用法同vector的pop_back或clear

判断

使用empty()，用法同vector的empty()

删除

使用erase，用法同vector中的erase
